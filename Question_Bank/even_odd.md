# Identify if given number is a **EVEN** or **ODD** number

### **isEven**
```javascript
let isEven = (x) => x % 2 === 0;
console.log(isEven(2)); // true

console.log(isEven(3)); // false
```

### **isOdd**
```javascript
let isOdd = (x) => x % 2 !== 0;
console.log(isOdd(2)); // false

console.log(isOdd(3)); // true
```
