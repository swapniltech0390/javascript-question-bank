# How to Create, Read & Delete a Cookie

## Creating a cookie
```javascript
document.cookie = "username=Swapnil";
```
## Reading a cookie
```javascript
let my_cookie = document.cookie;
```
## Deleting a cookie
```javascript
document.cookie = "username=Swapnil; expires=Mon, 20 Mar 2023 00:05:30 GMT"; 
```
